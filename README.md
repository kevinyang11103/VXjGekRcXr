# 停车场管理系统 java543

## 项目概述

停车场管理系统是基于SpringBoot和Vue的前后端分离的Web网站。系统支持智能停车收费管理，提供完整的停车运营管理平台。

## 技术栈

- **架构**：B/S结构、MVC软件架构、前后端分离
- **后端**：SpringBoot、MyBatis、Lombok、SaToken、Jwt
- **前端**：Vue、Element UI、Axios、ECharts
- **数据库**：MySQL
- **开发环境**：IDEA、JDK、Maven、Node

## 功能模块

### 系统角色

- 普通用户
- 管理员

### 普通用户功能

1. 注册：提供系统注册入口，确保用户可以登录系统
2. 登录：保障系统的安全访问
3. 首页：展示个性化欢迎界面，提供快捷导航，实时时间显示
4. 个人中心：用户可以查看和修改个人信息
5. 车位预订：允许用户预订可用车位
6. 缴费管理：管理账户余额和停车费用支付

### 管理员功能

1. 登录：专用的管理员登录入口，具备更高权限
2. 首页：系统概览、数据统计、快捷管理功能
3. 用户信息管理：全面管理系统所有用户
4. 车位信息管理：统一管理停车场所有车位
5. 停泊车辆管理：管理停车场内停泊的车辆信息
6. 车辆进出管理：记录和管理车辆进出详情
7. 车位费用设置：设置和管理车位收费标准
8. 登录日志查询：查看用户登录日志，确保系统安全

## 系统亮点

- **前后端分离**：基于SpringBoot+Vue开发，前后端分离使得前端响应快，后端逻辑清晰，便于模块化维护
- **多角色权限管理**：普通用户和管理员拥有不同的功能权限，使用JWT认证和路由守卫确保数据安全，构建完整的权限控制体系
- **智能化管理**：系统涵盖停车场运营全流程，从车位管理到用户服务，支持多角色协作，提升运营效率，实现停车场智能化管理
- **数据可视化**：集成ECharts图表组件，提供停车数据的可视化展示，为数据分析及决策提供直观支持

## 运行环境

- 后端运行环境：JDK 1.8+、Maven 3.6+
- 前端运行环境：Node.js 12+
